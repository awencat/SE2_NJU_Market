## 任务 2：AI 工具选型报告

### 一、AI 编程助手

#### 候选工具对比

| 评估维度       | GitHub Copilot                                             | OpenAI Codex（API）                                          |
| -------------- | ---------------------------------------------------------- | ------------------------------------------------------------ |
| **功能**       | 行内/函数级代码补全、注释生成代码、多语言支持、PR 描述生成 | 项目级上下文理解、自然语言生成完整代码文件、可自定义 Prompt 工作流 |
| **易用性**     | ⭐⭐⭐⭐⭐ 直接集成 VS Code，零配置即用                         | ⭐⭐⭐ 需调用 API，须自行搭建工作流或使用平台封装               |
| **成本**       | 学生可通过 GitHub Education 免费申请                       | 按 Token 计费，小规模使用成本可控                            |
| **团队熟悉度** | 高（VS Code 用户均已熟悉插件生态）                         | 中（需学习 API 使用与 Prompt 工程）                          |
| **上下文范围** | 当前文件 + 相邻已打开文件                                  | 可传入完整项目结构和多文件内容                               |

#### ✅ 选型决策

- **选用 GitHub Copilot**：日常编码主力工具，VS Code 内无缝使用，补全效率高，学生免费。
- **选用 OpenAI Codex**：用于架构级任务，如生成模块骨架、跨文件重构，需要项目级上下文时调用。

#### ❌ 未选原因参考（其他工具）
- **Cursor**：整体 IDE 替换成本较高，团队已统一使用 VS Code，迁移收益不明显。
- **通义灵码**：功能与 Copilot 高度重叠，国内访问有优势但团队成员已习惯 Copilot 工作流。

---

### 二、AI 对话工具

#### 候选工具对比

| 评估维度       | 豆包（字节跳动）                           | Google Gemini                                           |
| -------------- | ------------------------------------------ | ------------------------------------------------------- |
| **功能**       | 文档生成、需求分析、多轮对话、中文理解优秀 | 超长上下文（学生版免费 1M Token）、代码分析、多模态支持 |
| **易用性**     | ⭐⭐⭐⭐⭐ 国内直接访问，微信/网页均可用        | ⭐⭐⭐⭐ 需 Google 账号，学生账号可解锁更高配额             |
| **成本**       | 免费（基础版）                             | 学生账号免费且上下文极长                                |
| **团队熟悉度** | 高（日常已使用）                           | 中（需注册 Google 账号）                                |
| **上下文长度** | 约 128K Token                              | **1M Token**（学生版）                                  |
| **最适用场景** | 中文需求文档撰写、日常沟通记录整理         | 大文档分析、代码库整体理解、跨文件问答                  |

#### ✅ 选型决策

- **选用豆包**：用于日常中文文档撰写、会议记录、用例描述，访问便捷无障碍。
- **选用 Gemini**：用于需要超长上下文的任务，如整份需求文档分析、架构评审问答。

#### ❌ 未选原因参考（其他工具）
- **ChatGPT**：免费版上下文较短，Plus 版需付费，性价比低于学生免费的 Gemini。
- **文心一言**：中文能力接近豆包，但豆包产品体验更轻量，团队更熟悉。
