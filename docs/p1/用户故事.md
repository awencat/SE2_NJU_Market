👤 买家角色（10个）
US-01
作为一位经常在不同校区上课的学生，我想要按校区或宿舍区筛选商品，以便快速找到离我最近的可面交物品。

US-02
作为一位担心交易安全的学生，我想要看到卖家的实名认证标识（如学号认证），以便判断对方是否可信。

US-03
作为一位教材需求集中的学生，我想要按“教材/教辅”分类浏览商品，以便快速找到下学期需要的课本。

US-04
作为一位时间不固定的学生，我想要在商品详情页直接看到卖家的校区和宿舍区，以便判断是否方便线下取货。

US-05
作为一位想要讨价还价的学生，我想要通过平台内置的即时聊天功能联系卖家，以便不暴露个人微信也能沟通。

US-06
作为一位曾被“放鸽子”的学生，我想要查看卖家的历史交易评价和信用分，以便判断对方是否靠谱。

US-07
作为一位有特定需求但没人卖的学生，我想要发布“求购”信息，以便让有闲置的同学主动联系我。

US-08
作为一位想捡漏的学生，我想要按“价格从低到高”排序商品，以便优先看到最便宜的选择。

US-09
作为一位同时关注多个商品的学生，我想要收藏感兴趣的商品，以便在个人中心统一查看和对比。

US-10
作为一位担心买到问题商品的学生，我想要在交易完成后对卖家进行打分和文字评价，以便帮助后来的买家避坑。

🧑‍💼 卖家角色（7个）
US-11
作为一位毕业季需要清空宿舍的学生，我想要一键拍照+填写描述+定价发布商品，以便快速上架大量闲置物品。

US-12
作为一位不想被频繁问重复问题的卖家，我想要在商品发布时填写新旧程度、使用时长、瑕疵说明，以便减少不必要的沟通。

US-13
作为一位希望更快卖出商品的学生，我想要将商品同步发布到“赠送/交换专区”，以便如果卖不掉可以免费送人或换物。

US-14
作为一位跨校区上课的卖家，我想要在发布时选择“可跨校区代送”选项，以便扩大潜在买家范围。

US-15
作为一位曾被恶意差评的卖家，我想要对买家的交易行为进行反向评价，以便反映买家的爽约或不诚信行为。

US-16
作为一位卖贵重物品（如电脑、平板）的学生，我想要主动出示学号认证信息，以便让买家更信任我。

US-17
作为一位不想被骚扰的卖家，我想要在商品售出后手动下架或自动下架，以便不再收到新的咨询。

🛠️ 管理员/平台角色（3个）
US-18
作为一位平台管理员，我想要审核被举报的商品和用户，以便及时下架违规内容或封禁不良账号。

US-19
作为一位平台运营者，我想要查看用户的信用分分布和纠纷率，以便识别高风险用户并优化平台规则。

US-20
作为一位平台开发者，我想要记录用户的搜索和浏览行为（脱敏），以便分析热门商品类别并优化推荐逻辑。
